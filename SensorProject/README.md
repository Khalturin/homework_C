# SensorLogViewer

## Описание

Программа состоит из следующих файлов:
- main.c - Точка входа, содержит функцию main, с обработкой параметров командной строки
- sensor.h - файл описания структуры датчика и прототипов функий для работы с динамическим массивом этих структур
- sensor.c - файл с реализациями функций работы с динамическим массивом структур датчика
- temp_function.h - файл прототипов функция для работы с массивом структур датчика
- temp_function.c - файл с реализациями функций для работы с массивом структур датчика
- Makefile - файл инструкция для утилиты сборки make

Программа позволяет вывести статистику по каждому месяцу и году:
• среднемесячная температура
• минимальная температура в текущем месяце
• максимальная температура в текущем месяце
Вывести статистику за год:
• среднегодовая температура
• минимальная температура
• максимальная температура

## Установка 

Для установки находясь в дирректории проекта необходимо выполнить:
`make all`, при этом будет созданы 2 дирректории:
- `objects` - содержащая объектные файлы
- `build` - содержащая исполняемые файлы проекта.

## Использование

### Ключи командной строки

• -h Описание функционала приложения. Список ключей, которые обрабатывает данное приложение и их назначение.
• -f <ﬁlename.csv> входной файл csv для обработки.
• -m <номер месяца> если задан данный ключ, то выводится только статистика за указанный месяц.
• -y <год> если задан данный ключ, то выводится статистика за указанный год.
• -d выводится отладочная информация (прим. вся таблица). Используется только в качестве первого агрумента командной строки.

### Примеры использования
`./build/sensor -d -f source_data/temperature_small.csv` - выведет всё содержимое таблицы, находящейся в файле `temperature_small.csv`
`./build/sensor -f source_data/temperature_small.csv -m 7` - выведет только данные минимальной, максимальной и среднемесячной температуры из таблицы, находящейся в файле `temperature_small.csv`
`./build/sensor -f source_data/temperature_small.csv -m 7 -y 2021` - выведет только данные минимальной, максимальной и среднемесячной температуры, а также минимальной, максимальной и среднегодовой за 2021 год из таблицы, находящейся в файле `temperature_small.csv`
